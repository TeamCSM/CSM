# CSM

이 프로젝트는

1. 상업적 목적으로 개발된 프로젝트가 아닙니다.

2. 스프링 5계층 레이어 및 xyz식으로 구성되어있습니다.

3. 5명의 개발자가 만든 프로젝트입니다.

간략소개

1. 초급 개발자, 디자이너, 이제막 시작하는 취준생, 공모전 참가자 등 서로 서로가 필요한 사람들을 매칭시켜주는 사이트입니다.

2. 본인의 홈페이지 및 여러 물건들을 판매 할 수 있는 사이트 입니다.

구성

1. 소개
 - 프로젝트 간략 구성
 - 팀 소개

2. 매칭
 - 디자이너 매칭
 - 개발자 매칭

3. 커뮤니티
 - 자유게시판
 - 지식In
 - 출석체크

4. 포인트샵
 - 샵 메인, 등록, 판매, 리스트
 - 장바구니, 상품내역서 등

5. 고객센터
 - 공지사항
 - 자주찾는 질문
 - 1:1 문의

6. 마이페이지
 - 기본정인 정보
 - 회원탈퇴
 - 포인트 충전/반환
 - 개인 통계자료

7. 관리자
 - 홈페이지 전반 통계
 - 기타 게시판 관리

그 이외의 자세한 정보는 ppt참조
